Question-1: What are the different ways to select an element in DOM?

In JavaScript, you can select DOM elements in a few main ways. Use getElementById("id") for a unique ID, getElementsByClassName("class") for a group of elements with the same class, and getElementsByTagName("tag")for all elements of a certain type like paragraph. The modern and most flexible methods are querySelector("selector") for a single match and querySelectorAll("selector") for multiple matches using CSS-style selectors.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

In JavaScript, innerHTML, innerText, and textContent are used to get or set content of an element, but they work differently. innerHTML returns or sets the HTML inside an element (including tags).innerText returns or sets only the visible text, respecting CSS styles like display: none. textContent returns or sets all text inside the element, including hidden text, but without any HTML formatting.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where instead of adding event listeners to multiple child elements, you add a single event listener to their parent element. This works because events bubble up from the child to the parent, allowing the parent to detect and handle events from its children. It makes the code more efficient and easier to manage, especially when elements are added dynamically.

Question-4: What is event bubbling in the DOM?

IEvent bubbling is the process in the DOM where an event starts from the target element (the one that was actually clicked or interacted with) and then bubbles up through its parent elements until it reaches the root document. For example, if you click a span inside a div, the event first fires on the span, then on the div, and continues up the DOM tree. This bubbling behavior is what makes features like **event delegation** possible.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, you can create, add, and remove elements dynamically using DOM methods. To create a new element, you use document.createElement("tag") and can set its content with innerText or innerHTML. To add it to the page, you use appendChild or append on a parent element. To remove an element, you can either call element.remove() directly or use removeChild on its parent. These methods let you modify the webpage content on the fly without refreshing.
